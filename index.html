Le-co native grid

Minimalistic framework for native CSS Grid based on 12col grids

Just grab this little FW and give it a go.

Note #1: Even though this is a grid system in traditional sense of the word you might be familiar and comfortable with don't forget, that this application is really just the tip of the iceberg of options Native CSS grids provide.

Note #2: Don't forger to tweak variables to suite your needs.

Note #3: You might need to enable some flags in your browser to get this working.

This work was inspired by: CSS Grid Layout - creating complex grids

How this works?

To enable grid layout add class .grid to continaer. Then by using correct class combinations your can pick where you grid cell should being (starting column/row line) and to where it should span (end line). For example, if you want you cell to start at first column line and first row line you'd used classes .gc-s1 ("gc" as in grid column and "s" as in start) and .gr-s1 and to end at the end of third column on second line you'd add .gc-e4 and .gr-e2.

Gotchas: When assigning rows, you can leave it up to browser or you can add class .vertical to .grid to have rows with minimal 10px of height. At this point, this class is mostly usefull to check if you layed out cells correctly and that you are not skipping rows by having too high cells (which is fine, but it may lead to confusion when reading classes on cells).

Keep in mind, that his project is not final or perfect. It will however allow you to familiarise with yourself with native CSS grids via industry standard 12col grid.
