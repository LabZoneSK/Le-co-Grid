Le-co native grid

Minimalistic framework for native CSS Grid Layout based on 12col grids

Star Fork Download

This grid system uses well known concept of 12 col grid to demonstrate principles of native CSS Grid Layout, however native grids can do so much more and I encourage you to explore them further.

Note #1: You might need to enable some flags in your browser to get this working.

This work was inspired by: CSS Grid Layout - creating complex grids

How this works?

To enable grid layout add class .grid to continer. In case you want gutter, make it .grid.gutter. Then by using correct class combinations your can pick where you grid cell should being (starting column/row line) and to where it should span (end line). For example, if you want you cell to start at first column line and first row line you'd used classes .gc-s1 ("gc" as in grid column and "s" as in start) and .gr-s1 and to end at the end of third column on second line you'd add .gc-e4 and .gr-e2.

Gotchas: When assigning rows, you can leave it up to browser or you can add class .vertical to .grid to have 12 rows with minimal 1px of height. At this point, this class is mostly usefull to check if you layed out cells correctly and that you are not skipping rows by having too high cells (which is fine, but it may lead to confusion when reading classes on cells).

Tl;dr

Classnames define where grid cells start and end (in both column and row direction)

Example: .gc-s1 .gc-e3 .gr-s1 .gr-e2 will translate into cell starting at column grid line 1 and, ending at column grid line 3, starting at row line 1 and ending at row line 2.
