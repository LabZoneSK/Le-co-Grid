Le-co native grid

Minimalistic framework uses well known concept of 12 col grid to demonstrate principles of native CSS Grid Layout.

Star Fork Download

How this works?

Class .grid trigger grid layout. Add .gutter for gutter. If you add .vertical you'll get rows with 1px for layout debugging. Classnames on grid cells define where grid cells start and end (both in column and row direction).

Example: .gc-s1 .gc-e3 .gr-s1 .gr-e2 will translate into cell starting at column grid line 1 and, ending at column grid line 3, starting at row line 1 and ending at row line 2.

Notes

You might need to enable some flags in your browser to get this working.
