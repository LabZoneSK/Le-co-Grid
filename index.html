Le-co native grid

Minimalistic framework for native CSS Grid based on 12col grids

Native CSS grids are cool to use and learn. Just grab this little FW and give it a go.

Note #1: Even though this is a grid system in traditional sense of the word you might be familiar and comfortable with don't forget, that this application is really just the tip of the iceberg of options Native CSS grids provide.

Note #2: Don't forger to tweak variables to suite your needs.

Note #3: You might need to enable some flags in your browser to get this working.

This work was inspired by: CSS Grid Layout - creating complex grids
